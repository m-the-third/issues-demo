# square-demo

A simple CLI tool that computes the square of a number.

## Usage

```sh
uv run main.py <number>
```

### Examples

```sh
$ uv run main.py 5
5.0 squared is 25.0

$ uv run main.py -3
-3.0 squared is 9.0
```

## Development

### Run tests

```sh
uv run pytest
```
